Парные и Одиночые Теги

  1. Парные теги, необходмо закрывать:

    1. <h1-h6>: теги заголовков.
    2. <p>: тег абзаца.
    3. <a>: тег ссылки.

  2. Одиночные теги, закрываются сами:

    1. <br>: тег новой строки.
    2. <hr>: тег горизонтальной линии.
    3. <img>: тег изображения.

Блочные и Строчные Элементы

  1. Блочные теги, переходят на новую строку и занимают её полностью:

    1. <h1-h6>: теги заголовков.
    2. <p>: тег абзаца.
    3. <ol> и <ul>: нумерованные и маркированные списки.
    4. <li>: пункт списка.
    5. <header>,<main> и <footer>: семантические контейнеры.
    6. <nav>: навигационный блок.
    7. <hr>: тег горизонтальной линии.

  2. Строчные теги, не переходят на новую строку и занимают только нужную им ширину (по содержимому):

    1. <br>: тег новой строки.
    2. <img>: тег изображения.
    3. <a>: тег ссылки.

Изменения поведения через CSS:

Является ли блок строчным или блочным можно поменять через свойство display в CSS.

  1. "display: block" - превратит элемент в блочный
  2. "display: inline" - превратит элемент в строчный
  3. "display: inline-block" - превратит элемент в строчно-блочный, что превратит его в блок, но не будет занимать всю строчку.

Стилизация CSS и Атрибуты Тегов

Глобальные Атрибуты:

  1. id: Уникальный идентификатор элемента.
  2. class: Класс элемента (для CSS и JavaScript).
  3. style: Встроенные CSS-стили.
  4. title: Подсказка при наведении курсора.

Атрибуты для <img>:

  1. src: Путь к изображению.
  2. alt: Альтернативный текст, поясняет что находится в изображение (нужно для SEO, ботов и программ чтения с экрана).
  3. width: Ширина изображения, которую можно настроить напрямую, без CSS.
  4. height: Высота изображения, которую можно настроить напрямую, без CSS.

Атрибуты для <a>:

  1. href: Адрес перехода.
  2. target: Где открыть ссылку (_blank, _self).
  3. download: Скачать файл по ссылке, работает только с одного и того же домена.

Стилизация CSS:

Свойства:

  1. color: Цвет текста.
  2. background-color: Цвет фона.
  3. margin: Внешние отступы.
  4. padding: Внутренние отступы.
  5. width: Цвет текста.
  6. height -color: Цвет фона.
  7. font-family: Внешние отступы.
  8. font-size: Внутренние отступы.
  9. text-align: Цвет текста.
  10. vertical-align: Цвет фона.
  11. display: Внешние отступы.
  12. border-style: Внутренние отступы.
  13. border-color: Цвет текста.
  14. border-width: Цвет фона.
  15. margin: Внешние отступы.
  16. padding: Внутренние отступы.

Дополнительно:

  1. px, %, vw, vh, rem, em: Единицы измерения.
  2. /* */: Комментарии в CSS.
